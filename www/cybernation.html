    Coding in a language that we already know, is cliche. What if we learn a new one and start coding then and there ? You'll be given a choice of languages, in which you haven't coded earlier (probably), and your task will be to transform the given code into that language.
    RULES :
    1. Each team can have maximum 2 participants.
    2. Cross-college teams are allowed.
    3. Each team has to transform the given code from a well-known language like C to an esoteric language(eg. LOLCODE, brainfuck etc).
    4. Each team can select any esoteric language from the set of languages allowed.
    5. Teams are allowed to refer to the given documentation ONLY.
    6. Use of Internet is prohibited.
    7. Judgment Criteria - Team that converts maximum number of codes in minimum time.
    9. Total questions - 5.
